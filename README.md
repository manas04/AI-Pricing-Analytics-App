# AI Pricing Analytics App

A Streamlit-based pricing analytics application built with Python to analyze revenue, margin, and demand patterns.

## Features
- Upload CSV dataset
- View KPI dashboard
- Analyze pricing metrics
- Estimate price elasticity
- Simulate pricing scenarios

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
